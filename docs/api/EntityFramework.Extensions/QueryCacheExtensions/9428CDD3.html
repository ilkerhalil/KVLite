QueryCacheExtensions.

FromSlidingCacheFirstOrDefaultAsync<TEntity>(IQueryable<TEntity>, TimeSpan, IEnumerable<string>) Method

Summary

Returns the first element of the query; if possible from the cache, otherwise the query is materialized and the result cached before being returned. Query is cached as a "sliding" value, that is, value will last as much as specified in given interval and, if accessed before expiry, its lifetime will be extended by the interval itself.
