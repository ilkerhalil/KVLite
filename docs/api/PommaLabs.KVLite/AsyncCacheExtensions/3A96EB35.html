AddTimedAsync<TVal>(IAsyncCache, string, TVal, Instant, IList<string>, CancellationToken) Method

Summary

Adds a "timed" value with given key and default partition. Value will last until the specified time and, if accessed before expiry, its lifetime will _not_ be extended.
